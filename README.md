# GopherAI
GopherAI is a multimodal AI assistant project built with Go, Gin, and Vue, integrating AI chat, image recognition, user authentication, and session management. It supports streaming responses and combines backend APIs with a modern frontend interface for interactive user experiences.

GopherAI 是一个基于 Go、Gin 和 Vue 构建的多模态 AI 助手项目，集成了 AI 聊天、图像识别、用户认证和会话管理功能。它支持流式响应，并将后端 API 与现代化的前端界面相结合，提供交互式用户体验。
